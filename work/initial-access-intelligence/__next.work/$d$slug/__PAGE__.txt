## **Python Security Toolkit**

Developed a collection of **Python-based security tools** to simulate and analyze real-world cyber attack scenarios within a **controlled lab environment**.

- Built scripts to simulate common network attack techniques such as **traffic interception, device discovery, and packet analysis**, helping understand how systems can be exploited.
- Utilized **Scapy** for low-level packet crafting and inspection, enabling hands-on exploration of how data flows across networks.
- Created a **basic reverse shell** for controlled remote command execution during testing scenarios.
- Implemented simple **traffic analysis logic** to identify unusual or suspicious network behavior during simulations.

> Focus: **Cybersecurity fundamentals · Network behavior · Practical attack simulation**

## **Vulnerable Boot2Root Machine**

Designed and developed a **custom Boot2Root (B2R) machine** to simulate real-world attack scenarios, focusing on **initial access, enumeration, and privilege escalation** within a controlled lab environment.

- Crafted a vulnerable environment to practice **system enumeration and service discovery**, guiding users through identifying attack surfaces.
- Implemented an **SSRF (Server-Side Request Forgery) vulnerability** to demonstrate how internal services can be accessed and exploited.
- Designed restricted shell conditions requiring exploitation of **Linux binaries and misconfigurations** to achieve privilege escalation.
- Structured the machine as a guided learning experience, reinforcing practical understanding of **attack chains from initial access to root compromise**.

> Focus: **Enumeration · SSRF exploitation · Linux privilege escalation · Boot2Root methodology**
