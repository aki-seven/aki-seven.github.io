## **Python Security Toolkit**

Developed a collection of **Python-based security tools** to simulate and analyze real-world cyber attack scenarios within a **controlled lab environment**.

- Built scripts to simulate common network attack techniques such as **traffic interception, device discovery, and packet analysis**, helping understand how systems can be exploited.

- Created a **basic reverse shell** for controlled remote command execution during testing scenarios.
- Implemented simple **traffic analysis logic** to identify unusual or suspicious network behavior during simulations.

> Focus: **Cybersecurity fundamentals · Network behavior · Practical attack simulation**

## **IA-Intel: Intelligent Automated Enumeration Engine**

Developed a collection of **Python-based, configuration-driven security tools** to automate reconnaissance and service enumeration within a **black-box penetration testing environment**.

* Built a **phased execution engine** to orchestrate port scanning, service detection, and background enumeration in a structured pipeline.
* Utilized **Nmap and modular tool integrations (smbmap, enum4linux, Hydra, etc.)** for deep service-level analysis and validation.
* Created a **service-context model** to isolate and manage intelligence per port, enabling scalable and parallel enumeration.
* Implemented an **evidence-based validation system** to convert raw tool outputs into confirmed findings with confidence and severity scoring for faster enumeration.

> Focus: **Automation · Cybersecurity · Enumeration · Intelligence Analysis · System Design**
